Short Bex API
=============

API run on the server to facilitate the smart contract style payment channel we provide as an escrow service to lenders. Built with Python 3 and hug.rest.

Simply run the app and navigate to localhost:8000 to see the API documentation.

Usage
=====

Python 3 is required.

```
# pip3 install hug
$ hug -f api.py
```
